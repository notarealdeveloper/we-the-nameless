\Chapter{21}

\Verse{1}
{
	\hP{}
}
{
	\eP{
		And YHWH said to Moses, “Say to the priests, the sons of Aaron, and say
to them: One shall not become impure for a person among his people\\
	}
}
{
}

\Verse{2}
{
	\hP{}
}
{
	\eP{
		except for his relative who is close to him: for his mother and for his
father and for his son and for his daughter and for his brother\\
	}
}
{
}

\Verse{3}
{
	\hP{}
}
{
	\eP{
		and for his virgin sister who is close to him, who has not been a
husband’s; he may become impure for her.\\
	}
}
{
}

\Verse{4}
{
	\hP{}
}
{
	\eP{
		He shall not become impure as a husband among his people, for him to be
desecrated.\\
	}
}
{
}

\Verse{5}
{
	\hP{}
}
{
	\eP{
		They shall not make a bald place on their head and shall not shave the
corner of their beard and not make a cut in their flesh.\\
	}
}
{
}

\Verse{6}
{
	\hP{}
}
{
	\eP{
		They shall be holy to their God and shall not desecrate their God’s
name, because they are bringing forward YHWH’s offerings by fire, their
God’s bread, so they shall be a holy thing.\\
	}
}
{
}

\Verse{7}
{
	\hP{}
}
{
	\eP{
		They shall not take a woman who is a prostitute and desecrated, and
they shall not take a woman who is divorced from her husband, because he
is holy to his God.\\
	}
}
{
}

\Verse{8}
{
	\hP{}
}
{
	\eP{
		And you shall regard him as holy because he is bringing forward your
God’s bread. He shall be holy to you because I, YHWH, who makes you
holy, am holy.\\
	}
}
{
}

\Verse{9}
{
	\hP{}
}
{
	\eP{
		And the daughter of a man who is a priest who will become desecrated,
to whore: she is desecrating her father. She shall be burned in fire.\\
	}
}
{
}

\Verse{10}
{
	\hP{}
}
{
	\eP{
		“And the priest who is the most senior of his brothers, on whose head
the anointing oil will be poured and who has filled his hand to wear the
clothes, he shall not let loose the hair of his head and not tear his
clothes\\
	}
}
{
}

\Verse{11}
{
	\hP{}
}
{
	\eP{
		and he shall not come to any dead persons; he shall not become impure
for his father and for his mother.\\
	}
}
{
}

\Verse{12}
{
	\hP{}
}
{
	\eP{
		And he shall not go out from the holy place, so he will not desecrate
his God’s holy place, because the crown with his God’s anointing oil is
on him. I am YHWH.\\
	}
}
{
}

\Verse{13}
{
	\hP{}
}
{
	\eP{
		“And he shall take a woman in her virginity.\\
	}
}
{
}

\Verse{14}
{
	\hP{}
}
{
	\eP{
		A widow and a divorcée and a desecrated woman, a prostitute: he shall
not take these; he shall take none except a virgin from his people.\\
	}
}
{
}

\Verse{15}
{
	\hP{}
}
{
	\eP{

make him holy.”\\
	}
}
{
}

{
	\hP{}
}
{
	\eP{

{
}

}

	}
}
{
}

\Verse{19}

	\hP{}
}

}

	\hP{}
}
{
	\eP{

scabbed or scurvied or has crushed testicles.\\
	}
}
{
}

{
	\hP{}
}
{

injury in him. He shall not come near to bring forward his God’s bread.\\
	}
}
{
}

holies.\\
	}
}
{
}
