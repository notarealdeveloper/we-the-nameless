\Chapter{7}

\Verse{1}
{
	\hJ{ויאמר יהוה לנח בא אתה וכל ביתך אל התבה כי אתך ראיתי צדיק לפני בדור הזה}
}
{
	\eJ{
		And YHWH said to Noah,\\
		“Come, you and all your household, into an ark,\\
		for I’ve seen you as virtuous in front of me in this generation.\\
	}
}
{
}

\Verse{2}
{
	\hJ{מכל הבהמה הטהורה תקח לך שבעה שבעה איש ואשתו ומן הבהמה אשר לא טהרה הוא שנים איש ואשתו}
}
{
	\eJ{
		Of all the pure animals, take seven pairs,\\
		man and his woman;\\
		and of the animals that are not pure,\\
		two, man and his woman.\\
	}
}
{
}

\Verse{3}
{
	\hJ{גם מעוף השמים שבעה שבעה זכר ונקבה לחיות זרע על פני כל הארץ}
}
{
	\eJ{
		Also of the birds of the skies seven pairs,\\
		male and female,\\
		to keep seed alive on the face of the earth.\\
	}
}
{
}

\Verse{4}
{
	\hJ{כי לימים עוד שבעה אנכי ממטיר על הארץ ארבעים יום וארבעים לילה ומחיתי את כל היקום אשר עשיתי מעל פני האדמה}
}
{
	\eJ{
		Because in seven more days I’ll rain on the earth,\\
		forty days and forty nights,\\
		and I’ll wipe out all the substance that I’ve made\\
		from on the face of the earth.”\\
	}
}
{
}

\Verse{5}
{
	\hJ{ויעש נח ככל אשר צוהו יהוה}
}
{
	\eJ{
		And Noah did according to all that YHWH had commanded him.\\
	}
}
{
}

\Verse{6}
{
	\hBookOfRecords{ונח בן שש מאות שנה והמבול היה מים על הארץ}
}
{
	\eBookOfRecords{
		And Noah was six hundred years old\\
		when the flood was, water on the earth.\\
	}
}
{
}

\Verse{7}
{
	\hJ{ויבא נח ובניו ואשתו ונשי בניו אתו אל התבה מפני מי המבול}
}
{
	\eJ{
		And Noah and his sons and his wife\\
		and his sons’ wives with him\\
		came to the ark from before the waters of the flood.\\
	}
}
{
}

\Verse{8}
{
	\hP{מן הבהמה הטהורה ומן הבהמה אשר איננה טהרה ומן העוף וכל אשר רמש על האדמה}
}
{
	\eP{
		Of the animals that were pure\\
		and of the animals that were not pure,\\
		and of the birds and every one that creeps on the ground,\\
	}
}
{
}

\Verse{9}
{
	\hP{שנים שנים באו אל נח אל התבה זכר ונקבה כאשר צוה אלהים את נח}
}
{
	\eP{
		they came by twos to Noah, to the ark,\\
		male and female,\\
		as God had commanded Noah.\\
	}
}
{
}

\Verse{10}
{
	\hJ{ויהי לשבעת הימים ומי המבול היו על הארץ}
}
{
	\eJ{
		And seven days later\\
		the waters of the flood were on the earth.\\
	}
}
{
}

\Verse{11}
{
	\hP{בשנת שש מאות שנה לחיי נח בחדש השני בשבעה עשר יום לחדש ביום הזה נבקעו כל מעינת תהום רבה וארבת השמים נפתחו}
}
{
	\eP{
		In the six hundredth year of Noah’s life,\\
		in the second month,\\
		in the seventeenth day of the month,\\
		on this day all the fountains of the great deep were split open,\\
		and the apertures of the skies were opened.\\
	}
}
{
}

\Verse{12}
{
	\hJ{ויהי הגשם על הארץ ארבעים יום וארבעים לילה}
}
{
	\eJ{
		And there was rain on the earth,\\
		forty days and forty nights.\\
	}
}
{
}

\Verse{13}
{
	\hP{בעצם היום הזה בא נח ושם וחם ויפת בני נח ואשת נח ושלשת נשי בניו אתם אל התבה}
}
{
	\eP{
		In this very day Noah came,\\
		and Shem and Ham and Yaphet, Noah’s sons,\\
		and Noah’s wife and his sons’ three wives with them to the ark,\\
	}
}
{
}

\Verse{14}
{
	\hP{המה וכל החיה למינה וכל הבהמה למינה וכל הרמש הרמש על הארץ למינהו וכל העוף למינהו כל צפור כל כנף}
}
{
	\eP{
		they and all the wild animals by their kind\\
		and all the domestic animals by their kind\\
		and all the creeping animals that creep on the earth by their kind\\
		and all the birds by their kind,\\
		all fowl, all winged things.\\
	}
}
{
}

\Verse{15}
{
	\hP{ויבאו אל נח אל התבה שנים שנים מכל הבשר אשר בו רוח חיים}
}
{
	\eP{
		And they came to Noah, to the ark,\\
		by twos of all flesh\\
		in which was the breath of life,\\
	}
}
{
}

\Verse{16}
{
	\hP{והבאים זכר ונקבה מכל בשר באו כאשר צוה אתו אלהים}
	\hJ{ויסגר יהוה בעדו}
}
{
	\eP{
		and those that came were male and female;\\
		some from all flesh came,\\
		as God had commanded him.\\
	}
	\eJ{
		And YHWH closed it for him.\\
	}
}
{
}

\Verse{17}
{
	\hJ{ויהי המבול ארבעים יום על הארץ וירבו המים וישאו את התבה ותרם מעל הארץ}
}
{
	\eJ{
		And the flood was on the earth for forty days,\\
		and the waters multiplied and raised the ark,\\
		and it was lifted from the earth.\\
	}
}
{
}

\Verse{18}
{
	\hJ{ויגברו המים וירבו מאד על הארץ ותלך התבה על פני המים}
}
{
	\eJ{
		And the waters grew strong\\
		and multiplied very much on the earth,\\
		and the ark went on the face of the waters.\\
	}
}
{
}

\Verse{19}
{
	\hJ{והמים גברו מאד מאד על הארץ ויכסו כל ההרים הגבהים אשר תחת כל השמים}
}
{
	\eJ{
		And the waters had grown very, very strong on the earth,\\
		so they covered all the high mountains\\
		that are under all the skies.\\
	}
}
{
}

\Verse{20}
{
	\hJ{חמש עשרה אמה מלמעלה גברו המים ויכסו ההרים}
}
{
	\eJ{
		Fifteen cubits above,\\
		the waters grew stronger,\\
		and they covered the mountains.\\
	}
}
{
}

\Verse{21}
{
	\hP{ויגוע כל בשר הרמש על הארץ בעוף ובבהמה ובחיה ובכל השרץ השרץ על הארץ וכל האדם}
}
{
	\eP{
		And all flesh that creep on the earth—\\
		of the birds and of the domestic animals\\
		and of the wild animals\\
		and of all the swarming creatures that swarm on the earth,\\
		and all the humans—expired.\\
	}
}
{
}

\Verse{22}
{
	\hJ{כל אשר נשמת רוח חיים באפיו מכל אשר בחרבה מתו}
}
{
	\eJ{
		Everything that had the breathing spirit of life in its nostrils,\\
		everything that was on the ground, died.\\
	}
}
{
}

\Verse{23}
{
	\hJ{וימח את כל היקום אשר על פני האדמה מאדם עד בהמה עד רמש ועד עוף השמים וימחו מן הארץ וישאר אך נח ואשר אתו בתבה}
}
{
	\eJ{
		And He wiped out all the substance that was on the face of the earth,\\
		from human to animal to creeping thing and to bird of the skies,\\
		and they were wiped out from the earth,\\
		and just Noah and those who were with him in the ark were left.\\
	}
}
{
}

{

}
{
	\eP{
		And the water grew strong on the earth\\
		a hundred fifty days.\\
	}
}
