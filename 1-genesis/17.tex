\Chapter{17}

\Verse{1}
{
	\hP{ואברם בן תשעים שנה ותשע שנים וירא יהוה אל אברם ויאמר אליו אני אל שדי התהלך לפני והיה תמים}
}
{
	\eP{
		And Abram was ninety years and nine years old, and YHWH appeared to

	}
}
{
}

\Verse{2}
{
	\hP{ואתנה בריתי ביני ובינך וארבה אותך במאד מאד}
}
{
	\eP{
		and let me place my covenant between me and you, and I’ll make you
		very, very numerous.”\\
	}
}
{
}

\Verse{3}
{
	\hP{ויפל אברם על פניו וידבר אתו אלהים לאמר}
}
{
	\eP{
		And Abram fell on his face, and God spoke with him, saying,\\
	}
}
{
}

\Verse{4}
{
	\hP{אני הנה בריתי אתך והיית לאב המון גוים}
}
{
	\eP{
		“I: here, my covenant is with you, and you’ll become a father of a mass
		of nations.\\
	}
}
{
}

\Verse{5}
{
	\hP{ולא יקרא עוד את שמך אברם והיה שמך אברהם כי אב המון גוים נתתיך}
}
{
	\eP{
		And your name will not be called Abram anymore, but your name will be
		Abraham, because I’ve set you to be a father of a mass of nations.\\
	}
}
{
}

\Verse{6}
{
	\hP{והפרתי אתך במאד מאד ונתתיך לגוים ומלכים ממך יצאו}
}
{
	\eP{
		And I’ll make you very, very fruitful and make you into nations. And
		kings will come out of you.\\
	}
}
{
}

\Verse{7}
{
	\hP{והקמתי את בריתי ביני ובינך ובין זרעך אחריך לדרתם לברית עולם להיות לך לאלהים ולזרעך אחריך}
}
{
	\eP{
		And I’ll establish my covenant between me and you and your seed after
		you for their generations as an eternal covenant, to become God to you
		and to your seed after you.\\
	}
}
{
}

\Verse{8}
{
	\hP{ונתתי לך ולזרעך אחריך את ארץ מגריך את כל ארץ כנען לאחזת עולם והייתי להם לאלהים}
}
{
	\eP{
		And I’ll give you and your seed after you the land where you’re
		residing, all the land of Canaan, as an eternal possession, and I’ll
		become a God to them.”\\
	}
}
{
}

\Verse{9}
{
	\hP{ויאמר אלהים אל אברהם ואתה את בריתי תשמר אתה וזרעך אחריך לדרתם}
}
{
	\eP{
		And God said to Abraham, “And you: you shall observe my covenant, you
		and your seed after you through their generations.\\
	}
}
{
}

\Verse{10}
{
	\hP{זאת בריתי אשר תשמרו ביני וביניכם ובין זרעך אחריך המול לכם כל זכר}
}
{
	\eP{
		This is my covenant that you shall observe between me and you and your
		seed after you: every male is to be circumcised among you.\\
	}
}
{
}

\Verse{11}
{
	\hP{ונמלתם את בשר ערלתכם והיה לאות ברית ביני וביניכם}
}
{
	\eP{
		And you shall be circumcised at the flesh of your foreskin, and it
		will become a sign of a covenant between me and you.\\
	}
}
{
}

\Verse{12}
{
	\hP{ובן שמנת ימים ימול לכם כל זכר לדרתיכם יליד בית ומקנת כסף מכל בן נכר אשר לא מזרעך הוא}
}
{
	\eP{
		And at eight days old every male shall be circumcised among you
		through your generations: homeborn or purchased with money from any
		foreigner who is not from your seed.\\
	}
}
{
}

\Verse{13}
{
	\hP{המול ימול יליד ביתך ומקנת כספך והיתה בריתי בבשרכם לברית עולם}
}
{
	\eP{
		Your homeborn and the one purchased with your money will be
		circumcised. And my covenant will become an eternal covenant in your
		flesh.\\
	}
}
{
}

\Verse{14}
{
	\hP{וערל זכר אשר לא ימול את בשר ערלתו ונכרתה הנפש ההוא מעמיה את בריתי הפר}
}
{
	\eP{
		And an uncircumcised—a male the flesh of whose foreskin will not be
		circumcised—that person will be cut off from his people. He has broken
		my covenant.”\\
	}
}
{
}

\Verse{15}
{
	\hP{ויאמר אלהים אל אברהם שרי אשתך לא תקרא את שמה שרי כי שרה שמה}
}
{
	\eP{
		And God said to Abraham, “Sarai, your wife: you shall not call her
		name Sarai, because her name is Sarah.\\
	}
}
{
}

\Verse{16}
{
	\hP{וברכתי אתה וגם נתתי ממנה לך בן וברכתיה והיתה לגוים מלכי עמים ממנה יהיו}
}
{
	\eP{
		And I’ll bless her, and I’ll also give you a son from her. And I’ll
		bless her, and she’ll become nations. Kings of peoples will be from
		her.”\\
	}
}
{
}

\Verse{17}
{
	\hP{ויפל אברהם על פניו ויצחק ויאמר בלבו הלבן מאה שנה יולד ואם שרה הבת תשעים שנה תלד}
}
{
	\eP{
		And Abraham fell on his face and laughed and said in his heart, “Will
		he be born to someone who’s a hundred years old?! And will Sarah,
		who’s ninety years old, give birth?!”\\
	}
}
{
}

\Verse{18}
{
	\hP{ויאמר אברהם אל האלהים לו ישמעאל יחיה לפניך}
}
{
	\eP{
		And Abraham said to God, “If only Ishmael will live before you.”\\
	}
}
{
}

\Verse{19}
{
	\hP{ויאמר אלהים אבל שרה אשתך ילדת לך בן וקראת את שמו יצחק והקמתי את בריתי אתו לברית עולם לזרעו אחריו}
}
{
	\eP{
		And God said, “But Sarah, your wife, is giving birth to a son for you.
		And you shall call his name Isaac. And I’ll establish my covenant with
		him as an eternal covenant for his seed after him.\\
	}
}
{
}

\Verse{20}
{
	\hP{ולישמעאל שמעתיך הנה ברכתי אתו והפריתי אתו והרביתי אתו במאד מאד שנים עשר נשיאם יוליד ונתתיו לגוי גדול}
}
{
	\eP{
		And I’ve listened to you about Ishmael. Here, I’ve blessed him, and
		I’ll make him fruitful and make him very, very numerous. He’ll father
		twelve chieftains, and I’ll make him into a big nation.\\
	}
}
{
}

\Verse{21}
{
	\hP{ואת בריתי אקים את יצחק אשר תלד לך שרה למועד הזה בשנה האחרת}
}
{
	\eP{
		But I’ll establish my covenant with Isaac, whom Sarah will bear for
		you at this appointed time in the next year.”\\
	}
}
{
}

\Verse{22}
{
	\hP{ויכל לדבר אתו ויעל אלהים מעל אברהם}
}
{
	\eP{

	}
}
{
}

\Verse{23}
{
	\hP{ויקח אברהם את ישמעאל בנו ואת כל ילידי ביתו ואת כל מקנת כספו כל זכר באנשי בית אברהם וימל את בשר ערלתם בעצם היום הזה כאשר דבר אתו אלהים}
}
{
	\eP{
		And Abraham took Ishmael, his son, and all his homeborn and everyone

		had spoken with him.\\
	}
}
{
}

		flesh of his foreskin,\\
	}
}
{
}

	}
}
{
}
