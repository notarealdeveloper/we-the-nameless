\Chapter{15}

\Verse{1}
{
	\hJ{אחר הדברים האלה היה דבר יהוה אל אברם במחזה לאמר אל תירא אברם אנכי מגן לך שכרך הרבה מאד}
}
{
	\eJ{
		After these things YHWH’s word came to Abram in a vision, saying,\\
		“Don’t fear, Abram. I’m a shield for you. Your reward is very much.”\\
	}
}
{
}

\Verse{2}
{
	\hJ{ויאמר אברם אדני יהוה מה תתן לי ואנכי הולך ערירי ובן משק ביתי הוא דמשק אליעזר}
}
{
	\eJ{
		And Abram said, “My Lord, YHWH, what would you give me when I go
		childless and my household is an acquired person!” (That is Damascus
		Eliezer.)\\
	}
}
{
}

\Verse{3}
{
	\hJ{ויאמר אברם הן לי לא נתתה זרע והנה בן ביתי יורש אתי}
}
{
	\eJ{
		And Abram said, “Here, you haven’t given me seed. And, here, a member
		of my household is taking possession of what is mine.”\\
	}
}
{
}

\Verse{4}
{
	\hJ{והנה דבר יהוה אליו לאמר לא יירשך זה כי אם אשר יצא ממעיך הוא יירשך}
}
{
	\eJ{
		And, here, YHWH’s word came to him, saying, “This one won’t take
		possession from you, but rather one who will come out of your in-sides:
		he will take possession from you.”\\
	}
}
{
}

\Verse{5}
{
	\hJ{ויוצא אתו החוצה ויאמר הבט נא השמימה וספר הכוכבים אם תוכל לספר אתם ויאמר לו כה יהיה זרעך}
}
{
	\eJ{
		And He brought him outside and said, “Look at the skies and count the
		stars—if you’ll be able to count them.” And He said to him, “That is how
		your seed will be.”\\
	}
}
{
}

\Verse{6}
{
	\hJ{והאמן ביהוה ויחשבה לו צדקה}
}
{
	\eJ{
		And he trusted in YHWH, and He considered it for him as virtue.\\
	}
}
{
}

\Verse{7}
{
	\hJ{ויאמר אליו אני יהוה אשר הוצאתיך מאור כשדים לתת לך את הארץ הזאת לרשתה}
}
{
	\eJ{
		And He said to him, “I am YHWH, who brought you out of Ur of the
		Chaldees to give you this land, to possess it.”\\
	}
}
{
}

\Verse{8}
{
	\hJ{ויאמר אדני יהוה במה אדע כי אירשנה}
}
{
	\eJ{
		And he said, “My Lord YHWH, how will I know that I’ll possess it?”\\
	}
}
{
}

\Verse{9}
{
	\hJ{ויאמר אליו קחה לי עגלה משלשת ועז משלשת ואיל משלש ותר וגוזל}
}
{
	\eJ{
		And He said to him, “Take a three-year-old heifer and a three-year-old
		she-goat and a three-year-old ram and a dove and a pigeon for me.”\\
	}
}
{
}

\Verse{10}
{
	\hJ{ויקח לו את כל אלה ויבתר אתם בתוך ויתן איש בתרו לקראת רעהו ואת הצפור לא בתר}
}
{
	\eJ{
		And he took all of these for Him and split them up the middle and set
		each half opposite its other half, but he did not split the birds.\\
	}
}
{
}

\Verse{11}
{
	\hJ{וירד העיט על הפגרים וישב אתם אברם}
}
{
	\eJ{
		And birds of prey came down on the carcasses, and Abram retrieved
		them.\\
	}
}
{
}

\Verse{12}
{
	\hJ{ויהי השמש לבוא ותרדמה נפלה על אברם והנה אימה חשכה גדלה נפלת עליו}
}
{
	\eJ{
		And the sun was about to set, and a slumber had come over Abram; and,
		here, a big, dark terror was coming over him.\\
	}
}
{
}

\Verse{13}
{
	\hJ{ויאמר לאברם ידע תדע כי גר יהיה זרעך בארץ לא להם ועבדום וענו אתם ארבע מאות שנה}
}
{
	\eJ{
		And He said to Abram, “You shall know that your seed will be an alien
		in a land that is not theirs, and they will serve them, and they will
		degrade them four hundred years.\\
	}
}
{
}

\Verse{14}
{
	\hJ{וגם את הגוי אשר יעבדו דן אנכי ואחרי כן יצאו ברכש גדול}
}
{
	\eJ{
		But I’ll judge the nation whom they’ll serve as well, and after that
		they’ll go out with much property.\\
	}
}
{
}

\Verse{15}
{
	\hJ{ואתה תבוא אל אבתיך בשלום תקבר בשיבה טובה}
}
{
	\eJ{
		And you: you’ll come to your fathers in peace. You’ll be buried at a
		good old age.\\
	}
}
{
}

\Verse{16}
{
	\hJ{ודור רביעי ישובו הנה כי לא שלם עון האמרי עד הנה}
}
{
	\eJ{
		And a fourth generation will come back here, because the Amor-ite’s
		crime is not yet complete.”\\
	}
}
{
}

\Verse{17}
{
	\hJ{ויהי השמש באה ועלטה היה והנה תנור עשן ולפיד אש אשר עבר בין הגזרים האלה}
}
{
	\eJ{
		And the sun was setting, and there was darkness, and here was an oven
		of smoke and a flame of fire that went between these pieces.\\
	}
}
{
}

{
	\hJ{ביום ההוא כרת יהוה את אברם ברית לאמר לזרעך נתתי את הארץ הזאת מנהר מצרים עד הנהר הגדל נהר פרת}
}

}
